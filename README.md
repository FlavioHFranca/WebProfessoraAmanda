# Este projeto está totalmente containerizado, com ele garante que rodará com as mesmas dependências em qualquer máquina.

### Requisitos

Antes de começar, você precisa ter instalado na sua máquina:

- [Git]
- [Docker Desktop]

**Para usuários Windows:**
Dê uma olhada ser o docker desktop está aberto e com o serviço rodando antes de executar os comandos.

---

### Passos para Execução

**1. Faça o clone o repositório:**

```bash
git clone
cd WebProfessoraAmanda
```
